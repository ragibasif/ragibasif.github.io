---
title: Clear the Lowest Set Bit
date: 2025-12-28
id: 1
---

To "clear the lowest set bit" means turn the lowest bit with a value of `1` to a
value of `0`. The formula is `n & (n - 1)`.

This is also used to check if a number is a power of 2. If a number is a power
of 2, then performing `n & (n - 1)` will return 0.
